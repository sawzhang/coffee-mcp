# 优惠发现与领券

帮助用户发现 Coffee Company 的活动和优惠券。

## 工具选择

| 场景 | 工具 |
|------|------|
| 查看活动日历 | `campaign_calendar` |
| 查看可领取的券 | `available_coupons` |
| 一键领取全部 | `claim_all_coupons` |

领券成功后引导用户去点单使用。
